float intensity = 1,
float duration = 0::5f,
AnimationCurve curve = default,

static

Shakes the inputted camera for a specified amount of time with a certain intensity following an animation curve.

For best results, don't move the camera directly via any other scripts. Camera can also be any transform, so you can shake other objects with these

Parameters
cameraCamera to apply screenshake to, default is Camera.main
intensityA multiplier for more screenshake
durationHow many seconds the screenshake lasts for
curveCurve used to apply screenshake
useRealtimeIf true, will use Time.unscaledDeltatime, instead of Time.deltaTime
