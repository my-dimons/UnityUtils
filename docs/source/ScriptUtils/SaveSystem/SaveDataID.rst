SaveDataID
================

**NAMESPACE:**
   `UnityUtils.ScriptUtils.SaveSystem`
     
The **SaveDataID** script is used in turn with other save systems to assist with save systems. This script holds data relating to save files, things like file name, its unique id (:doc:`SaveDataRegistry` thing), and the :doc:`ISaveData` instance.

Example Usage
-------------
SaveDataID's must be created via the :doc:`SaveDataRegistry` to work properly
