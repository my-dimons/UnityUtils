UIButtonQuitGame
==========

**NAMESPACE:**
   `UnityUtils.ScriptUtils.UI`
   
Usage
-------------
Add a 'UIButtonQuitGame' Script to a UI button and it will quit the game on click by default. You can customize what action will cause the game to quit as your needs suit.
